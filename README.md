### URL
lezin-react.vercel.app(lezin-react.vercel.app)

### Install
    npm start

### Story
기존 레진코믹스 사이트를 다르게 구성했습니다. 자체 json 파일을 만들어서 이미지 클릭시 세부 페이지로 넘어갈 수 있게 만들었습니다. 

### Skill
`react` `styled-compontents` `firebase` `context`
